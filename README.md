# Блокиратор звонков по префиксу

Приложение для Android, которое блокирует входящие звонки с номеров, начинающихся с заданного префикса. 
Разработано для защиты от спам-звонков и мошенников.

## Функционал

- Блокировка звонков по префиксу (первые 3 цифры после +7)
- Добавление и удаление префиксов
- Включение и выключение отдельных префиксов
- Светлая и темная тема оформления
- Онбординг при первом запуске
- Работает на Android 10 и выше

## Технологии

- Язык: Java
- Минимальная версия Android: 10 (API 29)
- Библиотеки:
  - Material Design 3
  - RecyclerView
  - Gson (для работы с JSON)
- Системный API: CallScreeningService

# Prefix Call Blocker

An Android application that blocks incoming calls from numbers starting with a specified prefix.
Developed to protect against spam calls and fraudsters.

## Features

- Call blocking by prefix (first 3 digits after +7)
- Add and remove prefixes
- Enable and disable individual prefixes
- Light and dark theme support
- Onboarding screen on first launch
- Works on Android 10 and above

## Technologies

- Language: Java
- Minimum Android version: 10 (API 29)
- Libraries:
  - Material Design 3
  - RecyclerView
  - Gson (for JSON processing)
- System API: CallScreeningService
